# Calendário de agendamento
Este projeto é uma aplicação de reserva de salas (backend + frontend). O backend fornece uma API REST para gerenciar salas e agendamentos (CRUD) e usa um banco em memória para facilitar o desenvolvimento, o frontend exibe um calendário por andares/salas que permite criar, editar e excluir agendamentos visualmente.

## Tecnologias usadas

* Backend: Spring Boot (Java), gerenciado com Maven.

* Banco de desenvolvimento: H2.

* Frontend: React + Vite.

## Como rodar
### Na pasta backend:

`mvn spring-boot:run`

### Na pasta frontend:

`npm install`

`npm run dev`
